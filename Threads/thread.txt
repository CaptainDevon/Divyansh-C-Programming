Q) what is Thread?
A) --> In every application there is a default thread which
       is maintained which is main()
   --> In this side we create other threads
   --> A thread is known as the light weight process. These 
       are used to achieve parallelism by dividing a process 
       multiple threads

   -->example:
       1)Chrome browser has multiple tabs can be having different 
         Threads.
       2)MS word must be using multiple threads ,one thread is to format 
            text, another thread is to process input(spell checker)


Q) What are the ways to create threads?
A) 1)Function Pointers
   2)Lambda Functions
   3)Member Functions
   4)Static Member Functions
